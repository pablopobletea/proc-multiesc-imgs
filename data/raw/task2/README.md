# Datos — Tarea 2

Esta carpeta contiene los volúmenes MRI (.nii) y las imágenes astronómicas (.png) utilizadas en la Tarea 2.

## Archivos

- `gt.nii`: volumen de referencia (ground truth)
- `streaking.nii`: reconstrucción degradada
- `reconNDI*`: reconstrucciones basadas en métodos NDI
- `recontv*`: reconstrucciones basadas en variación total (TV)
- `mask.nii`: máscara de evaluación
- `Moon.png`: imagen astronómica de alta frecuencia
- `EX3_01.png`: imagen de cielo profundo

## Nota

> Por razones de tamaño, estos archivos no se versionan en git (ver `.gitignore`).
